STOCK MARKET CHATBOT

This project is specializied chatbot for stock market. This chatbot only answers te quesitons for stock market. Otherwise, the chatbot answers like 'I am sorry, I was only trained to help with the stock market'.

Llamaindex was used in the project. Llama-3.2-1b was taken from hugginface for artificial intelligence training in the project. Streamlit was used for GUI design and frontend components.I used llamaindex to personalize the chatbot and provided a ready-made prompt.
Requirement libraries added inside of the requirements.txt. After dowloading the project on the local of the computer, you need to type 'cd my_env' in powershell. And type 'streamlit run ChatApp.py' in powershell for open the project.

Once open the project you need write the question you want to ask in the message bar and send it, the answer will appear on the screen.
